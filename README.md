# sentimental_analysis
Scraping  amazon reviews for one plus 6t. 

Required Packages :
  1) lxml  (html parser)
  2) Beautiful soup (library for web scraping) 
  3) requests 
  4) textblob  (used for sentimental analysis)
  5) pymongo


to Run :
  python program.py
